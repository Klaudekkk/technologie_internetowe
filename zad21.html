Zad 21

Napisz funkcję, która przyjmie jeden argument zawierające zdanie złożone z kilku słów. Następnie funkcja powinna podzielić to zdanie na słowa (ignorując znaki interpunkcyjne) i stworzyć tablicę słów. Tablicę tą należy wyświetlić. Następnie proszę stworzyć obiekt który wyświetli (w tagu < code >) ilość słów użytych w zdaniu. Np: Ala ma kota, a kot ma Alę -> [ala, ma, kota, a, kot, ma, alę] -> { ala: 1, ma: 2, kota: 1, a: 1, kot: 1, alę: 1}.
